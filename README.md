# MS Nuckem Modpack Source

This private repository is reserved for MS Nuckem modpack files, launcher reports, and release metadata.

Recommended flow for the customer:

1. Put client mods into `mods/`.
2. Keep config files in `config/` when needed.
3. Update `build.json` when the public launcher source is ready.
4. Do not upload private passwords or personal tokens.

The launcher can point to raw files from this repository after the final hosting contract is locked.
